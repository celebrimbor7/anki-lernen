# anki-lernen
Ich zeige euch, wie ich mit Anki lerne und gleichzeitig zeige ich euch meine Erfahrungen in HTML, CSS, JavaScript in Anki.

Ich bin in Sprachen lernen interessiert und habe mich mit der Grammatik befasst. Hauptsächlich werden Verben, Adverbien, Nomen und Adjektive gelernt. Daher habe ich Anki-Karten in HTML, CSS, JavaScript geschrieben mit dem Ziel, dass der Lernenden so wenig wie möglich eingeben muss, um grammatische Flexionen richtig angezeigt zu lernen.
