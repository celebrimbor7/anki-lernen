# Deklination

Nomen werden durch den grammatischen Kasus, Genus und Numerus bestimmt. In Anki ist der Artikel notwendig, um erstens die Deklination generell so einzuschränken, dass der Code einfacher, kürzer und (in den allermeisten Fällen) richtig funktioniert und zweitens der Numerus und Genus wiedergeben wird.

Warum nutze ich den Demonstrativum anstatt des Artikels? Weil es den Artikel in den meisten Sprachen nicht gibt, zumal sich der Artikel aus dem Demonstrativum abgeleitet hat. Daher wird der Demonstrativum genutzt, um die Deklination zu vereinfachen und schließlich den Code zu vereinfachen.

## Tschechisch 🇨🇿

### Wie dekliniere ich tschechische Nomen?

Siehe meine Grammatik_Deklination.xlsx. Der Artikel (eigentlich ist der ein Demonstrativum) ist eine Voraussetzung, dass der Code richtig ausgeführt wird. Zuerst wird der Numerus und Genus durch den Artikel (Nominativ Singular) und den Artikel (Nominativ Plural) bestimmt, danach wird die vereinfachte Deklination durch die Endung der Form (Nominativ Singular) bestimmt. In manchen Fällen sind die Formen: Nominativ Plural, Genitiv Singular und Genitiv Plural auch notwendig, um alles abzudecken, was für Ausnahmen / Regeln die tschechische Sprache hat.

Also, es gibt diese Formen:
1. Nominativ Singular
2. Nominativ Plural
3. Genitiv Singular
4. Genitiv Plural
5. Artikel (Nominativ Singular)
6. Artikel (Nominativ Plural)

Tschechisch kennt drei Ausnahmen / Regeln:
1. Erweichung
2. Eine Änderung -e ↔ -
3. Eine Änderung -ů- ↔ -o-

#### Erweichung
Die Erweichung ist die einzige Ausnahme / Regel, die der Code durch die Form (Nominativ Singular) bestimmen kann. Bei der Erweichung werden harte Konsonanten erweicht. Diese heißen weiche Konsonanten.
